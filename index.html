musique by tulskiy

100% Java audio player. Supports mp3, ogg, flac, wavpack, ape, wav, au, aiff. Builtin support for cue files, gapless playback, SHOUTCast/IceCast Radio, last.fm scrobbling.

Dependencies

OpenJDK 6 or Sun/Oracle JDK 6

Install

Just unpack and run musique.sh or musique.bat
